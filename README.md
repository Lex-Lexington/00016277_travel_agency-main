# 00016277_travel_agency

This project was created to improve the existing modules of the website of the travel agency **BEST HOLIDAYS TOUR**. Best Holidays Tour is a travel company of Uzbekistan, which was established in 2014.

Design inspired by: https://www.youtube.com/watch?v=TalusE29iqg&t=133s

## Live Website

[Visit the website](https://lex-lexington.github.io/00016277_travel_agency-main/index.html)
